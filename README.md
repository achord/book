## About This Book

This is an editorial working document, not a published page.

The author — a 47-year-old man writing to the young men in his life — has been drafting this book since May 2024. The material covers the life lessons he learned the hard way: financial literacy, identity, integrity, the power of starting early, communication, relationships, and how to navigate a world transformed by technology.

The book is addressed primarily to his son Corbin (and son Sam), and explicitly to young middle-class men who were not handed a foundation — who have to figure it out themselves. The author's four core principles are: **Autonomy**, **Consistency**, **Connection**, and **Time**.

This repository contains raw notes, dated fragments, outline drafts, GPT feedback, quotes, and partially developed chapters. The `book/` directory organizes these into a chapter-based structure. The `source-archive/` directory preserves all original files exactly as written.

## Source Material Notes

All original markdown files are preserved exactly as written in [`source-archive/`](./source-archive/). No source material has been deleted. The `book/` directory reorganizes and references this material without replacing it.

**Date preservation:** Every source file has a date in its filename (e.g., `2024-06-13 - Forward.md`). All dates are preserved in the corresponding subsection files in their `Source References` sections. Files with multiple writing sessions (notably `2024-06-13 - Forward.md`, which contains entries from June 2024 and April 2026) have all dates documented.

**Original filenames:** Original file names contain dates and are preserved in `source-archive/` without renaming.

**Typos and misspellings:** The original files contain numerous typos and misspellings (e.g., "Intergrity" for "Integrity", "especislly" for "especially", "Scott Gallaway" for "Scott Galloway"). These are preserved in the source archive and in quoted material. The note-file names in `source-archive/` are not corrected. Chapter headings and directory names use corrected spellings.
